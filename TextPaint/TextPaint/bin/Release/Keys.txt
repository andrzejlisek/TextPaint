Arrows, Home, End, PageUp, PageDown - move cursor
Tab - cursor style
Esc, F1, F2, F3, F4 - change state
F7, F8 - Save/reload file
F9 - Character selector
F12 - Quit

State 0 - Edit text - additional keys:
  Esc - writing direction
  Backspace - move cursor in opposite direction
  Insert/Delete - insert or delete
  Enter - change Insert/Delete mode
  
State 1,2 - Draw frame, character
  
State 3,4 - Draw rectangle or diamond - additional keys:
  W,S,A,D - Change size
  Q,E - Change cursor position within the rectangle/diamond
  1 - Change frame character set.
  2 - Draw frame using character set.
  3 - Draw frame using character selected to draw.
  4 - Fill using character selected to draw.
  Insert/Delete - insert or delete
  Enter - change Insert/Delete mode
  F4 (state 4 only) - change diamond shape variant

  I or Numpad plus - Vertical line
  K or Numpad minus - Horizontal line
  Space or Numpad 0 - Space (erase character)

  Frame corners, tees and cross using the following letters or numpad digits:
  T-Y-U  7-8-9
  | | |  | | |
  G-H-J  4-5-6
  | | |  | | |
  B-N-M  1-2-3
